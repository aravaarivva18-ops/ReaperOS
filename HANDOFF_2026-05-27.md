## 🚨 Важные ограничения

1. **venv обязателен** — глобальный `pytest`/`flask` не найдёт зависимости. Всегда `.reaper_venv/bin/pytest`
2. **db.sqlite в .gitignore** — не коммитить базу данных
3. **git worktree требует коммита** — нельзя создать worktree на unstaged изменениях
4. **Pre-commit hook активен** — коммит упадёт если тесты красные

## 📐 Архитектурные решения (ADR)

| # | Решение | Причина |
|---|---------|---------|
| 1 | SQLite вместо Redis | Нет внешних зависимостей, работает offline |
| 2 | ThreadPoolExecutor для параллелизма | Аналитики I/O-bound, GIL не мешает |
| 3 | launchd plist для watchdog | macOS-native, надёжнее cron |
| 4 | Detached HEAD в worktrees | Изоляция эксперимента без создания ветки |
